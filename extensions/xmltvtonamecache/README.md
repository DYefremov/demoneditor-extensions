## XMLTV to EPG name cache.

Enables the ability to add and match a service name from the favorites list with the required XMLTV identifier   
by dragging the desired service onto the list in the EPG tab and adding it to the name cache.

### Requires [View XMLTV cache](https://github.com/DYefremov/demoneditor-extensions/tree/main/extensions/viewxmltvcache) extension installed and [DemonEditor](https://github.com/DYefremov/DemonEditor) >= 3.14.4
